# ゲームプロジェクト

このフォルダには個々のゲームプロジェクトが格納されます。

## 新しいゲームの追加方法

1. `games/`ディレクトリに新しいフォルダを作成
2. `cargo new --name ゲーム名` でプロジェクトを初期化
3. ゲームのCargo.tomlに必要な依存関係を追加
4. 共通ライブラリ（`bevy-games-common`）を使用する場合は依存関係に追加

## ゲーム構造の例

```
games/
├── pong/           # Pongゲーム
├── snake/          # Snakeゲーム
├── breakout/       # Breakoutゲーム
└── platformer/     # プラットフォーマーゲーム
```

各ゲームは独立したCargoプロジェクトとして管理され、必要に応じて共通ライブラリを利用できます。
